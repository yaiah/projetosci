📖 Como usar este guia

Este guia foi construído para apoiar a disciplina na realização de diagnósticos participativos. Cada página apresenta conceitos, etapas e atividades práticas que podem ser adaptadas ao contexto específico do campus e da comunidade.

Os estudantes podem usar este material para atividades de sala de aula. Aqui encontrarão suporte metodológico para elaboração de diagnósticos de extensão.
